Version 0.4

作成日：2026-07-10

⸻

1. プロジェクト概要

プロジェクト名

Project Library

（開発コードネーム）

⸻

目的

無料でダウンロードできる脳トレ・知育プリントを公開し、

利用者が探しやすく、

運営者が続けやすい

「作品管理システム付き無料プリント図書館」

を作る。

⸻

運営目標

・スマホだけでも投稿できる

・投稿時間3分以内

・作品数が増えるほど資産になる

・広告収益で運営費以上を目指す

・長く運営できるサイトを作る

⸻

2. 設計思想

Project Libraryでは

「未来の自分が困らない」

を最優先にする。

開発ルール

・固定費はできるだけ0円

・設計書を先に更新する

・コードは設計書を見ながら作る

・利用者も運営者も使いやすい設計

・スマホだけでも運営可能

・同じことを二度入力しない

・将来の拡張が簡単

・作品数が1000件になっても管理しやすい設計

⸻

5. 対象者カテゴリ

・子供向け

・大人向け

・シニア・リハビリ

作品は複数カテゴリ所属可能。

6. タグ設計

タグは2種類。

固定タグ

作品ジャンル。

例

・ぬりえ

・迷路

・間違い探し

・シール貼り

・点つなぎ

・計算

・工作

将来追加可能。

⸻

自由タグ

作品内容。

例

・夏

・恐竜

・消防車

・海

・動物

・乗り物

将来追加可能。

⸻

タグ入力ルール

新規タグ入力時は

既存タグ候補を表示。

同じ意味のタグ作成を防ぐ。

固定タグも後から追加可能。

⸻

9. 広告方針

利用者の邪魔をしないことを最優先。

作品ページでは

・横長広告（ダウンロードボタン前）

・ページ最下部広告

を配置。

ダウンロード後には広告を表示しない。

将来的に表示回数に応じて広告配置を調整可能。

⸻
