PDF仕様（更新）

PDF下部余白へ

以下を共通表示する。

* © あたまのストレッチ プリント館
* 作品番号（PL-000001形式）
* サイトQRコード
* 「無料プリントはこちら」

QRコードは作品を邪魔しないサイズで右下へ配置する。

目的

* 印刷物からサイトへ戻れる導線
* 新しい利用者への案内
* 作品管理のしやすさ
